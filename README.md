# Tisseo Transport App

Dans le cadre de la pratique portant sur le langage javascript afin de consolider nos compétences en JS et en API REST.
Il nous a été demandé à travers un premier défi, de développer une application Web à partir des données publiées sur l’open data de Tisseo et accessibles à partir d’une API REST.

Cette application que vous trouverez sous vos yeux est une application web pour consulter les lignes et arrêts du réseau **Tisseo** (Toulouse). 
Développée en HTML, CSS et JavaScript, elle utilise l’API officielle de Tisseo pour afficher les données en temps réel.

---

## 📌 Fonctionnalités
- **Affichage des lignes** avec un simple clic.
- **Liste des arrêts** pour chaque ligne sélectionnée.
- **Design moderne** et responsive, avec animations fluides.
- **Loader** pendant le chargement des données.

---

## 🛠 Technologies utilisées
- **HTML5** : Structure de la page.
- **CSS3** : Style moderne avec animations et dégradés.
- **JavaScript** : Récupération et affichage des données via l’API Tisseo.
- **API Tisseo** : Données officielles sur les lignes et arrêts.
